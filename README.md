### Zero False Positive Detection bot.

TUBARÃO DB is a system at Roblox / Discord which detects inappopriate user such ERP activity.

You may know there are other system such Rotector or TASE(The All Seeing Eye), but all of them are not free from false positive.

TUBARÃO DB is completely different, here's how:

- Collects data from trusted games / servers, based on system-provided logs.
- Does not use selfbot. All data is inserted by human operators.
- No mass insert, only with clear evidence.

With this, we have only two status of player: Unknown, Inappopriate.

Our goal is creating trusted database without any false positive controversy, which could lead to slower detection, but that's our natural.
